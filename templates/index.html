xi.pe - "zippy" URL Shortener

URL Shortener

Shorten your URLs with our simple API or use the form on the right:

/api/urlpost?ttl=1d&url=https://example.com

TTL options:

  • 1d - 1 day (4 character code)
  • 1w - 1 week (5 character code)
  • 1m - 1 month (6 character code)

Then access your shortened URL at: xi.pe/[code]

API Usage

  • Shorten URL: /api/urlpost?ttl=[1d|1w|1m]&url=[encoded_url]
  • Access URL: /[code] (4-6 alphanumeric characters)
  • Test URL: /[code]?action=info (view info without redirecting)
  • Format Override: Add &format=[json|html] to any endpoint to override response format
  • Statistics: /stats
