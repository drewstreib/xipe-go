xi.pe - zippy text sharing

API Documentation

Share text or URLs programmatically with our JSON API:

Endpoint: POST /api/post
Content-Type: application/json

Share Text (default)

{"ttl":"1d","data":"your text here"}
Returns: {"status":"ok","url":"http://xi.pe/Ab3d"}

Share URL

{"ttl":"1d","data":"https://example.com","typ":"URL"}
Returns: {"status":"ok","url":"http://xi.pe/XyZ9"}

TTL Options: 1d (4 chars), 1w (5 chars), 1mo (6 chars)

Limits: Text up to 10KB, URLs up to 4KB
