URL Shortener API

Shorten URLs with JSON API or use the form on the right:

  • Shorten: POST /api/post with {"ttl":"1d","url":"https://example.com"}
  • Access: /[code] (shows info page with original URL)

TTL options: 1d (4 chars), 1w (5 chars), 1mo (6 chars)

Text Sharing Service

Store text snippets with JSON API or use the form on the right:

  • Store: POST /api/post with {"ttl":"1d","data":"your text here"}
  • Access: /[code] (shows data page with stored text)

Data limit: 10KB max. Same TTL options as URL shortener.
