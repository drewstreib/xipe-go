xi.pe text sharing

API Documentation

Endpoint: POST /
Content-Type: application/json
 
{"data":"your text here"}
Returns: {"status":"ok","url":"https://xi.pe/Ab3d"}
 
Optional TTL: "ttl":"[1d|1w|1mo]" (defaults to "1d")
Optional URL formatting: "typ":"URL"
Limits: Text up to 50KB, URLs up to 4KB
Python:
import requests
shorturl = requests.post('https://xi.pe/', json={"data": data}).json()['url']
